# Azul Sprinkler System

A multi-component project for an automated sprinkler controller.

## Project Structure

- **mobile/**: React Native (Expo) mobile application for controlling the system.
- **server/**: Backend API and logic (TBD).
- **firmware/**: ESP32 firmware projects for various devices.
- **shared/**: Shared types, constants, and assets.
- **docs/**: Architecture diagrams, wiring schematics, and API specifications.

## Getting Started

### Mobile App
```bash
cd mobile
npm install
npm start
```
